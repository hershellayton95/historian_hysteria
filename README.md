# historian_hysteria

## Requisiti  

- **Rust**

## Installation  

1. Clone the repository:  
   ```bash
   git clone https://github.com/tuo-utente/historian_hysteria.git
   cd historian_hysteria
   ```  

2. Complie the project:  
   ```bash
   cargo build --release
   ```  

## Usage  

Exec the program with  
```bash
./target/release/historian_hysteria
```
